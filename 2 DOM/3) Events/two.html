JavaScript by default:-

  • JavaScript is by default Synchronuos language(after one code execution it will execute another code)
  • JavaScript works on single thread i.e. it is Single Threaded language

Execution Context

  • Execute one line of code at a time
    • conlsole log-1
    • console log-2
    Each operation waits for the last one to complete before execution
  • Call Stack
  • Memory heap

Blocking Code vs Non Blocking Code

  • Blocking Code:-
    • Block the flow of the program
    • Read File Synchronuos

  • Non-Blocking Code:-
    • Do not block execution
    • Read File Asynchronuos

Blocking and Non blocking code both are important each one has it's own use case

Blocking code:- if user doing registration then we should wait until we write user data into database and then we can show user message that you have successfully registered
